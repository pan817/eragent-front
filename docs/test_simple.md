# P2P 分析接口测试用例集

> 每条用例标注执行路径：**DAG**（L1/L2 命中 → DAG 并行执行 + ReportAgent 汇总）、**ReAct**（L3 兜底 → Agent 自主调用工具）、**早退**（模板直答，零 LLM）、**Lookup**（DATA_LOOKUP 快捷路径，直调 query_* 工具）。
>
> 路由层级说明：
> - **L1**：关键词命中率评分（零延迟）
> - **L2**：Chroma 种子库语义匹配（毫秒级）
> - **L3**：LLM 分类 + 参数提取（0.5~2s）
